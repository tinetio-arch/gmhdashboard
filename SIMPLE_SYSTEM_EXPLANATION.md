## The Technical Part (You Don't Need to Understand This!)

**But if you're curious:** We use something called "MCP" (Model Context Protocol) to connect all your systems. Think of it like a universal translator that lets Heidi talk to Healthie, GHL, your database, etc.

Each system has a "server" that Heidi can talk to:
- Database Server - talks to your patient database
- Healthie Server - talks to Healthie API
- GHL Server - talks to GoHighLevel
- Email Server - sends emails
- Heidi Server - handles voice and notes

All these servers work together so Heidi can do anything you need!
